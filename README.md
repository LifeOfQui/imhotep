# Imhotep

## Installation
run yarn
- `yarn install`
  
## Usage
place files in specific folders:
- Images: `app/assets/images` 
- JS: `app/assets/scripts`
- CSS: `app/assets/styles`

run gulp task
- `gulp watch`
  
## Deployment
run gulp task
- `gulp build`
